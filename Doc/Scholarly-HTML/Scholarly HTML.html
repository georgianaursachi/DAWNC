Abstract

Aplicația web are rolul de a simula comportamentul unui mușuroi de furnici.

Cuvinte cheie

antvior, proiect, infoiasi, cliw, web

Descriere proiect/ Motivație

Nou: te poți alătura echipei noastre Antvior pentru a ne ajuta să dezvoltăm aplicația.

Această aplicație este încă în dezvoltare așa ca simte-te liber să te alături echipei noastre sau să ne trimiți ideile tale prin e-mail.

Un utilizator va putea folosi aplicația web onilne. Acesta poate alege cu cate furnici sa inceapa simularea(valoarea fiind cuprinsa intre 1 si 120), mancarea furnicilor din colonie, precum si cata mancare va fi pe harta. In simulator se va putea urmări evoluția societății de furnici. In plus, userul mai poate sa interactioneze cu furnicile, si anume, sa adauge/elimine manual furnici si mancare. Utilizatorul mai poate urmari si urma de feromoni lasata de furnici pe harta. Furnicile au si un grad de frica vis-a-vis de pozitia cursorului. De asemena, utilizatorul se poate informa despre viata, temperamentul furnicilor in pagina dedicata acesteia.

Aplicația are rol educațional, fiind utila pentru studiul minimal al comportamentului unei societăți de furnici.

Tehnologii utilizate

HTML5/CSS3

HTML5 este un limbaj pentru structurarea și prezentarea conținutului pentru World Wide Web. HTML5 introduce un număr de noi elemente și atribute care reflectă utilizarea tipică a unui site modern.

Am ales să folosim HTML5 deoarece:

  • interfața existentă document object model (DOM) este extinsă de fapt cu caracteristici noi, documentate. Sunt noi API-uri, cum ar fi elementul canvas pentru modul de desen 2D. (necesar pentru animarea furnicilor, plasarea mancarii pe harta, vizualizarea urmei de fermoni)
  • introduce un număr de noi elemente și atribute care reflectă utilizarea tipică a unui site modern.(cum ar fi <div> și de elemente inline (<span>), de exemplu <nav> - block de navigatie în site -, <footer>)

CSS (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. CSS3 reprezintă un upgrade ce aduce câteva atribute noi și ajută la dezvoltarea noilor concepte in webdesign.

Unul din marile plusuri ale CSS3 este modulul "Media queries", care faciliteaza scrierea usoara a codului pentru diferite tipuri de ecrane.

Javascript

JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul JavaScript din aceste pagini fiind rulat de către browser. Limbajul este binecunoscut pentru folosirea sa în construirea site-urilor web.

Am ales să folosim plain Javascript deoarece:

  • Este inteles de orice versiune de browser.
  • Faciliteaza manipularea elementelor DOM(Document Object Model), dar si "ascultarea" evenimentelor din cadrul paginei web, pentru a putea raspunde corespunzator.
  • Faciliteaza animatiile, dar si partea de algoritm a aplicatiei.(elemente de programare structurata/structuri de control: if, for, while, etc.)

Arhitectura aplicației

Front-end

Site-ul este o suma de fisiere html, css si javascript.

Site-ul este responsive, fiind facilitata astfel vizualizarea lui pe mai multe device-uri/ecrane.(Folosind Media queries)

Organizare fisiere:

Furnicile

Furnicile, mancarea si feromnonii sunt desenate pe un canvas(Element HTML5 folosit pentru a desena componente grafice intr-o pagina web).

Back-end

Site-ul nu necesita parte de server(back-end development), deoarece toata aplicatia este realizata la nivel de client.
