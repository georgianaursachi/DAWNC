Abstract

Aplicația web are rolul de a simula comportamentul unui mușuroi de furnici.

Cuvinte cheie

antvior, proiect, infoiasi, cliw, web

Descriere proiect/ Motivație

Nou: te poți alătura echipei noastre Antvior pentru a ne ajuta să dezvoltăm aplicația.

Această aplicație este încă în dezvoltare așa ca simte-te liber să te alături echipei noastre sau să ne trimiți ideile tale prin e-mail.

Un utilizator va putea folosi aplicația web onilne. El va porni simulatorul cu cel putin doua furnici, si va urmări evoluția societății de furnici. Va putea salva progresul la moment dat, fiind capabil sa îl reîncarce pentru a continua cu simulatorul din momentul in care l-a salvat.

Aplicația are rol educațional, fiind utila pentru studiul minimal al comportamentului unei societăți de furnici.

Tehnologii utilizate

HTML5/CSS3

HTML5 este un limbaj pentru structurarea și prezentarea conținutului pentru World Wide Web. HTML5 introduce un număr de noi elemente și atribute care reflectă utilizarea tipică a unui site modern.

CSS (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. CSS3 reprezintă un upgrade ce aduce câteva atribute noi și ajută la dezvoltarea noilor concepte in webdesign.

Javascript

JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul JavaScript din aceste pagini fiind rulat de către browser. Limbajul este binecunoscut pentru folosirea sa în construirea siturilor web.

Am ales să folosim Javascript deoarece:

  • Aplicația noastră web se bazează pe modele 3D.
  • Permite importarea lor într-o pagina web, in mod simplu.
  • Facilitează animațiile.

Maya

Maya este un program 3D de animație, modelare, simulare și randare. Util pentru animație, scene, grafică mișcare, realitate virtuală și crearea de caractere.

Am ales să folosim Maya deoarece dorim sa redam cât mai bine realitatea, folosind modele 3D. De asemena, Maya facilitează animarea obiectelor.
